# Folders

Folders help you organize elements and scripts in the Solution Explorer.

## Where Folders Can Be Created

| Location | Action | Purpose |
|----------|--------|---------|
| **Workspace** | Right-click → New Folder | Organize elements (buttons, timers, devices, etc.) |
| **Workspace folder** | Right-click → New Folder | Create subfolders |
| **Processes** | Right-click → New Folder | Organize scripts |
| **Processes folder** | Right-click → New Folder | Create subfolders for scripts |

## Folder Actions

Right-click a folder for:

- **Rename Folder** — Change the folder name
- **Delete Folder** — Remove the folder (elements and scripts move to the parent)

## Drag and Drop

You can drag elements and scripts into folders to reorganize. Folders can also be dragged to reorder or nest.
