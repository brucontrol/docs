# Dashboard Overview

The **Dashboard** is the main canvas where workspace elements appear. Each workspace has its own dashboard.

## What You See

- **Elements** — Each element (button, timer, chart, device control, etc.) renders on the canvas
- **Tabs** — When you have multiple workspaces, tabs appear above the dashboard; click to switch
- **Canvas** — Drag, resize, and arrange elements; zoom and pan for large layouts

## Modes

- **Edit mode** — Click elements to select; double-click to open the Edit Drawer. Hold Space to temporarily pan. On touch devices, drag on empty canvas to pan.
- **Pan mode** — Pan the canvas by dragging (no element selection). Pan is the default on first load.

Elements are added via the Solution Explorer context menu, not from the dashboard canvas.

## Element Interaction

Elements show real-time values. Elements with **User Control** enabled let you interact (toggle switches, buttons, sliders) directly from the dashboard. Processes run scripts and automation in the background.
