# WINC1500 Wi-Fi Considerations

Some interfaces use the Atmel WINC1500 Wi-Fi module (e.g., Arduino MEGA with WINC1500 shield, Adafruit Feather M0 WINC1500, Adafruit Grand Central M4 with WINC1500 shield). ESP32 and ESP8266 use built-in Wi-Fi, not WINC1500. Setup and configuration may differ from standard Ethernet interfaces.

For network setup steps, see [Firmware Installation and Setup](../hardware/firmware-installation#network-configuration-ethernetwi-fi-only).

For connectivity issues, see [Troubleshooting](./troubleshooting).
